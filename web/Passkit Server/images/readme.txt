In addition to the data, a passbook pass also includes images.
At least there habe to be the pictures icon.png, icon@2x.png and logo.png.
The following images will be recognized by the iOS6 Passbook App:
background.png
background@2x.png
icon.png
icon@2x.png
logo.png
logo@2x.png
strip.png
strip@2x.png
thumbnail.png
thumbnail@2x.png
